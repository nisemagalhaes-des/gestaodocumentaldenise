# Site Denise Soares — Gestão Documental (Versão Premium)

Arquivos principais:
- index.html
- style.css

Antes de publicar:
1. Abra o arquivo index.html.
2. Troque todos os links com `5500000000000` pelo número correto do WhatsApp.
   Formato: 55 + DDD + número. Exemplo: 5586999999999.
3. Troque `SEUINSTAGRAM` pelo usuário correto do Instagram.
4. Substitua os arquivos antigos do GitHub por esta nova versão.
5. Aguarde a atualização do GitHub Pages.
